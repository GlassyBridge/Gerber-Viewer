Gerber Viewer

Better that you use PC for this demo

How to use?

  • Load Gerber files using the file input at the bottom left corner.
  • Use mouse wheel to zoom in and out, click and drag to pan the view. (I think touch is supported too)
  • Use the buttons at the top center to switch between top, bottom and layers view.
  • Use the controls at the bottom center to zoom in, zoom out and reset the view.
  • In layers view only: Use the layers panel at the right side to toggle visibility of individual layers.

What did I make this project?

I made this project to serve as a simple tool for viewing Gerber files just like tracespace.io does. I wanted to make it unique by giving the user an option to generate permanent links to use for demo purposes for hardware projects on platforms like in SoM. I also wanted to add a 3d version which enables the user to place components on it that could also be shared with a permanent link.

Unfortunately, due to time constraints and my lack of skills in these kinds of developments, I decided to make a testable demo, with basic features, just before the end of SoM. So this project doesn't have those promised features but I plan to get to them even well after Summer of making ends.

For now, I hope you find this project useful and fun to use. P.S my gerber drawing function is not great so don't bully me on that (^-^)

You could support me to continue working on this project by voting for it on SoM :) Thanks in advance.
